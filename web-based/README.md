# remote:web-based

Aims to control devices synced through raspberry pi using hand gestures. This codebase is a case study for web-based gesture recognization using GEST.js

## WHAT'S IN

- [x] Gest.js
- [x] socket.io

### Installation

```sh
npm install
```

### Run

```sh
node src/index.js
```

or

```sh
npm start
```

### Features

- [x] Computer Vision - client based
- [ ] IoT
